# StockAnalysis

## Rajasthan Hackathon 4.0
### To run this application use the below two parameters from the command line. 
```
python main_stock_module.py values.csv
```
This Project is in early stages hence the GUI for this hasn't been made in the scheduled two days of hackathon.This uses **Stocker** module to take in the Previous sales and then represent the data in Graphs.
